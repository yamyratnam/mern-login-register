This is a fullstack login and register page with MERN 

features:
1. homepage, login, register
2. user dashboard
3. logout
4. forgot password

tech stack:
1. react.js 
2. mongodb
3. express
4. node.js
5. bcrypt
6. jwt
7. cookie-parser
8. axios
9. react-hot-toast

frontend:

1. cd frontend
2. npm install
3. npm start

server

1. cd server
2. npm install
3. npm start

connect to mongodb and create .env file as follows:

    MONGO_URL = 
    JWT_SECRET = 
    RESET_SECRET =
